## Mobile Development Standards

**Platform Integration:**
- Implement push notifications (FCM/APNs)
- Add biometric authentication (Face ID/Touch ID/fingerprint)
- Integrate with device cameras and sensors
- Handle deep linking and universal links
- Implement in-app purchases and subscriptions
- Manage app permissions properly

**Mobile UI/UX Implementation:**
- Follow iOS Human Interface Guidelines
- Implement Material Design on Android
- Create smooth page transitions and animations
- Handle keyboard interactions properly
- Implement pull-to-refresh patterns
- Support dark mode across platforms
- Implement native navigation patterns

**Performance Optimization:**
- Optimize app launch time (<2 seconds)
- Implement efficient list virtualization
- Optimize image loading and caching
- Minimize bridge calls (React Native)
- Use native animations when possible
- Profile and fix memory leaks
- Reduce app bundle size

**Cross-Platform Development:**
- Share business logic across platforms
- Platform-specific UI when needed
- Handle platform differences gracefully
- Test on both platforms regularly
- Optimize for platform-specific patterns
- Maintain native feel despite sharing code

**App Store Preparation:**
- Optimize app size and startup time
- Implement crash reporting and analytics
- Create App Store/Play Store assets
- Handle app updates gracefully
- Implement proper versioning
- Manage beta testing (TestFlight/Play Console)

**Performance Targets:**
- App launch time: <2 seconds cold start
- Frame rate: consistent 60fps (120fps on capable devices)
- Memory usage: <150MB baseline
- Battery impact: minimal
- Network efficiency: bundled requests
- Crash rate: <0.1%

**Technology Expertise:**
- iOS: Swift, SwiftUI, UIKit, Combine
- Android: Kotlin, Jetpack Compose, Coroutines
- Cross-Platform: React Native, Flutter, Expo
- Backend: Firebase, Amplify, Supabase
- Testing: XCTest, Espresso, Detox

**Mobile-Specific Patterns:**
- Offline-first architecture
- Optimistic UI updates
- Background task handling
- State preservation across app lifecycle
- Deep linking strategies
- Push notification patterns
